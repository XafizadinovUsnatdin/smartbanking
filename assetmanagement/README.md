## Smart Office Asset Management (Frontend)

Vite + React dashboard for SmartBanking/SmartOffice asset lifecycle management.

### Run (dev)

1) Make sure backend is running (Docker Compose in repo root):
   - `docker compose up --build`

2) Configure API base URLs:
   - `cp .env.example .env` (macOS/Linux) or `copy .env.example .env` (Windows)

3) Start UI:
   - `npm install`
   - `npm run dev`

UI: `http://localhost:5173/login`

Language: switch `UZ / EN` in the header.
